# 开发环境搭建

```js
vue init webpack demo1

// 按照提示输入即可。
// 为了学习方便， 采用完整构建，vue-router，ESLint，unit，e2e暂时全部不需要

// 完成以上交互命令，提示如下

   vue-cli · Generated "demo1".

   To get started:

     cd demo1  // 切换到demo1目录
     npm install // 安装依赖
     npm run dev // 启动服务

   Documentation can be found at https://vuejs-templates.github.io/webpack

```
